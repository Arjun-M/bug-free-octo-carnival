## [1.0.0] - 2024-05-20

### Added
- Initial release of IsoBox
- Secure JavaScript/TypeScript execution using `isolated-vm`
- Virtual filesystem with quota management and permissions
- Module resolution system with whitelist support and mocking
- Session management for persistent execution contexts
- Connection pooling for high-throughput scenarios
- Comprehensive test suite
- Full TypeScript support
- Streaming execution support
- Security logging and error sanitization
